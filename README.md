# una

## 包含的功能列表

### config 配置（toml）

### logger 日志（zap）

### gin-middleware Gin框架中间件
* access log
* recovery
* request ID

### project info 项目基础信息
* 项目名称
* 程序名称
* 项目相关目录：根目录、资源目录等

### tools 常用工具函数
* 文件下载
* 哈希函数：MD5
* 文件/目录相关函数：判断文件/目录是否存在
* AES 加解密
* 随机数
* IP 相关函数：ip2long，判断是否在指定 IP 段内，获取本机内网 IP
